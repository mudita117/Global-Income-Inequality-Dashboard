# 🌍 Global Income Inequality Dashboard (Power BI)

An interactive **Power BI dashboard** analyzing global income inequality, poverty, GDP, unemployment, and regional disparities across countries over time.  
This project uncovers **economic drivers, trends, and strategic insights** to support data-driven policy and business decisions.

---

# 📌 Project Overview

Income inequality remains one of the biggest global economic challenges.  
This dashboard explores:

- How inequality varies across **regions and income groups**
- Relationship between **GDP, poverty, unemployment & inequality**
- **Long-term trends** in global inequality
- **Strategic insights** for decision making

---

# 🎯 Objectives

- Analyze global inequality using key economic indicators  
- Identify regions and countries with the **highest income gaps**  
- Understand **economic drivers** behind inequality  
- Track **historical trends and socioeconomic impact**  
- Provide **executive-level insights**

---

# 🛠️ Tools & Technologies

- 📊 **Power BI** – Data modeling & visualization  
- 🔄 **Power Query** – Data cleaning & transformation  
- 🧮 **DAX** – Measures & calculations  
- 📁 **Excel Dataset** – Source data  
- 🔖 **Navigation Buttons** – Dashboard storytelling  
- 📂 **File Formats:** `.pbix`, `.png`

# 📊 Dashboard Structure

The dashboard contains **4 interactive pages**:

---

## 🟦 Page 1 — Executive Overview

### Key KPIs
- Average Gini Index → **0.38**
- Average GDP per Capita → **20.57K**
- Average Poverty Rate → **20.14%**
- Average Unemployment Rate → **7.92%**
- Average Income Gap → **0.12**

### Key Visuals
- Income gap by region  
- Population distribution by region  
- Top 10 countries with highest inequality  
- Inequality category distribution  

### Key Insight
Global inequality remains **moderate but widespread**, with certain regions significantly more affected.

---

## 🟩 Page 2 — Economic Drivers of Inequality

### Visual Analysis
- GDP Category vs Income Gap  
- GDP per Capita vs Gini Index  
- Poverty vs Inequality by Income Group  
- Income Gap Heatmap by Region  

### Key Findings
- Economic growth **does not always reduce inequality**  
- Higher poverty strongly correlates with **higher inequality**  
- Upper-middle income countries show **significant income gaps**

---

## 🟨 Page 3 — Inequality Trends & Socioeconomic Impact

### Visual Analysis
- Inequality vs Poverty over time  
- Global inequality trend by region  
- Unemployment vs Inequality by region  
- Urbanization vs Poverty  

### Key Findings
- Inequality peaked around the **early 2000s**  
- Regions show **different inequality trajectories**  
- Higher urbanization links with **lower poverty**

---

## 🟥 Page 4 — Executive Insights & Conclusions

### 📌 Economic Development
Economic growth alone is **not enough** to reduce inequality.

### 📌 Poverty & Inequality
High poverty strongly associates with **higher inequality**.

### 📌 Regional Disparities
Inequality varies due to **structural economic differences**.

### 📌 Income Distribution
Income remains **highly concentrated** globally.

# 📂 Dataset Description

| Column | Description |
|---|---|
| Country | Country name |
| Region | Geographic region |
| Year | Time period |
| GDP per Capita | Economic output per person |
| Poverty Rate | % population below poverty line |
| Unemployment Rate | % unemployed population |
| Gini Index | Income inequality measure |
| Income Gap | Income disparity indicator |
| Urban Population % | Urbanization level |
